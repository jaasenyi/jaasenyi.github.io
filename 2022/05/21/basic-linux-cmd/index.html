linux基础命令补充

目录

多用户操作系统

/ -> home -> 用户

命令

Part1:（终端基础）

命令 解释
pwd 查看当前所在文件夹
man command 查看command帮助信息
.开头 隐藏文件，ls -a 显示隐藏文件
.&.. .当前目录、..上一级目录
cd - 最近两次工作目录间切换
mkdir -p 连续创建多及目录。例：mkdir -p a/b/c
rm -r -f -r递归删除目录，删除文件夹必须加此参数。-f强制删除
tree 树状图列出文件目录结果，-d：只显示目录不显示文件

Part2:（文件操作）

命令 解释
cp -i 文件重复时，提示是否覆盖当前文件
cp -r 递归复制目录
mv -i 提示是否覆盖当前文件
cat & more cat一次 显示所有内容。more分屏显示
more 空格：显示下一屏；enter键：滚动一行；b：回滚一屏；f：前滚一屏；q：退出。
cat 查看文件、创建文件、文件合并、追加文件内容等
cat -b -n 查看并输出编号；-n所有行加编号；-b空行不加编号

Part 3:（文件内容）

命令 解释
grep 对文件文本内容进行查找
grep -n 显示匹配同时指定行号
grep -v 显示不包含匹配文本的所有行（求反）
grep -i 忽略大小写
^a 行首匹配-搜索行首为a的行
ke$ 行尾匹配-搜索行尾为ke的行
> 输出，覆盖文件原有内容。可结合echo使用
>> 追加，将内容追加到已有文件的末尾。可结合echo使用

Part 4:（远程）

命令 解释
管道：｜ 一个命令的输出，可通过管道，作为另一个命令的输入
ssh [-p port] user@remote remote是远程机器地址可ip/域名
scp 远程拷贝文件，地址格式同ssh，指定端口大写的-P
scp -P port 01.py user@remote:Desktop/01.py 将本地当前目录下的01.py文件复制到远程家目录下的Desktop/01.py
scp -P port user@remote:Desktop/01.py 01.py 将远程家目录下的Desktop/01.py文件复制到本地当前目录下的01.py
scp -r 可传送文件夹

Part 5:（SSH高级）

ssh配置信息保存在用户家目录.ssh目录下

1、免密码登录

  • 配置公钥

    执行ssh-keygen,生成SSH公钥。

  • 上传公钥到服务器

    （让远程服务器记住我们的公钥）

    执行ssh-copy-id -p port user@remote

2、配置别名

自定义名字代替-p port user@remote

~/.ssh/config里追加以下内容：

Host zhangsan

	Hostname 目标主机ip

	User 目标主机名字

	Port 22

保存后 ssh zhangsan 即可登录，scp同样可以使用

Part 6:（ls命令&通配符）

命令 解释
ls -l -l 列表方式显示文件详细信息
ls -lh -h配合-l人性化显示文件大小
ls -a ls -a 显示隐藏文件
通配符 解释
* 代表任意个数字符。例：ls 1*：列出1开头所有文件
？ 用法参考*，但只能代表一个字符
[] 可匹配字符组中任意一个
[abc] 匹配a、b、c任意一个
[a-f] 匹配a到f任意一个字符

ls -l 详细信息

  • 权限：第一个字符若为d，表示目录。
  • 硬链接数：通俗讲，有多少方式可访问到当前目录。
  • 拥有者：家目录下文件/目录拥有者通常是当前用户。
  • 组
  • 大小
  • 时间
  • 名称

Part 7:（用户/组）

1、chmod

  • chmod 可修改用户/组对文件/目录的权限

  • chmod +/- rwx 文件名｜目录名

    • 例：chmod +r 1.py 给文件1.py增加可读权限

      此方式会一次性修改·拥有者/组和其他用户·的权限

  • chmod -R 777 文件名｜目录名 递归修改文件权限；777->（用户、组、其他用户 均为rwx）

2、组

命令 解释
groupadd 添加组
groupdel 删除组
cat/etc/group 确认组信息（组信息保存在/etc/group）
chgrp -R 目标组名 文件名/目录名 递归修改文件/目录所属组

​ 2.1、usermod

  • ​ usermod 可设置用户的主组/附加组和登录shell

  • ​ 主组：通常在新建用户时指定，在etc/passwd的第4列GID对应的组

  • ​ 附加组：在etc/group中最后一列表示该组的用户列表，用于指定用户的附加权限。

    命令 解释
    usermod -g 组 用户名 修改用户的主组
    usermod -G 组 用户名 修改用户的附加组（重新登录生效）
    usermod -s /bin/bash 修改用户的登录shell

    useradd添加用户无sudo权限，可将其添加附加组到sudo

3、用户

命令 解释
useradd -m -g 组 新建用户名 -m：自动建立用户家目录
-g：指定用户所在组，否则会建立一个同名组
passwd 用户名 设置用户密码、普通用户直接passwd修改自己的账户密码
userdel -r 用户名 -r： 自动删除用户家目录
`cat /etc/passwd grep 用户名`

​ 3.1、查看用户信息

命令 解释
id 查看用户UID和GID信息
who 查看当前所有登录的用户列表，以及用户的来源
whoami 查看当前登录用户的账户名
su - 用户名 切换用户；使用-切换到用户家目录否则路径不变；`exit退出到上级用户

​ 3.2、passwd文件

  • 用户名
  • 密码（x，表示加密的密码）
  • UID（用户标识）
  • GID（组标识）
  • 用户全名或本地账号
  • 家目录
  • 登录使用的shell （ubuntu默认使用dash）

4、which

  • /etc/passwd：保存用户信息的文件
  • /usr/bin/passwd：用于修改用户密码的程序
  • witch命令可以查看执行命令的所在位置

bin 和sbin

在linux中，绝大多数可执行文件保存在/bin、 /sbin、/usr/bin、/user/sbin

  • **/bin**（binary）二进制执行文件目录，主要用于具体应用
  • **/sbin**（system binary）系统管理员专用二进制代码存放目录，主要用于系统管理
  • **/usr/bin**（user commands for applications）后期安装的一些软件
  • **/usr/sbin**（super user commands for applications）超级用户的一些管理程序

5、chown

修改拥有者

chown 用户名 文件名｜目录名

Part 8:（系统信息）

命令 解释
date 当前系统时间
cal 查看日历-y查看一年日历
df -h disk free 显示磁盘剩余空间
du -h[目录名] disk usage显示目录下文件大小

8.1、进程信息

命令 解释
ps aux process status 查看进程详细情况
a显示所有包括其他用户的进程；u显示进程详细状态；x显示没有控制终端的进程
top 动态显示运行中进程且排序；q退出
kill[-9]进程代号 终止进程代号的进程，-9表示强行终止

使用'kill'，终止'root'身份开启的进程可能导致系统崩溃

8.2、find

命令 解释
find[路径] -name "文件" 查找指定路径下的文件；可搭配通配符使用；省略路径表示当前路径查找

8.3、软链接&硬链接

命令 解释
ln -s 被链接的源文件 链接文件 建立文件软链接，类似windows快捷方式
  • 没有-s选项建立的是一个硬链接文件
  • 源文件要使用绝对路径，移动链接文件后仍正正常使用
  • linux中，文件名和文件数据分开存储。
  • ls -l可查看硬链接数。

8.4、打包压缩

  • Windows 常用 rar
  • Mac 常用 zip
  • Linux 常用tar.gz

打包：（tar）

命令 解释
tar -cvf 打包文件.tar 被打包的文件/路径 打包文件
tar -xvf 打包文件.tar 解包文件
tar选项：c、x、v、f c生成档案文件，创建打包文件；x解开档案文件；v列出归档解档详细过程，显示进度；f指定文件名称，须放最后，跟.tar文件。

压缩：（gzip）

  • gzip压缩tar打包后的文件，扩展名一般用xxx.tar.gz
  • tar命令参数-z可调用gzip，实现压缩解压缩
命令 解释
tar -zcvf 打包文件.tar.gz 被压缩的文件/路径 压缩文件
tar -zxvf 打包文件.tar.gz 解压缩文件
tar -zxvf 打包文件.tar.gz -C 目标路径 解压缩到指定路径
-C 解压到指定目录，且目录必须存在。

压缩：（bzip2）

  • bzip2压缩tar打包后的文件，扩展名一般用xxx.tar.bz
  • tar命令参数-j可调用bzip2，实现压缩解压缩
命令 解释
tar -jcvf 打包文件.tar.bz2 被压缩的文件/路径 压缩文件
tar -jxvf 打包文件.tar.bz2 解压缩文件
-C 解压到指定目录，目录必须存在。
