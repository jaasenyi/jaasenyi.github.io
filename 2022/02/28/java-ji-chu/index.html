Java基础语法(Part-1)

Java基础

学习：狂神说Java(秦疆老师)视频所记笔记

1、注释

  • 单行注释
//单行注释
  • 多行注释
/*
	多行注释
	多行注释
	多行注释
*/
  • 文档注释
//Javadoc：文档注释  /**  */
/**
*@Description HelloWorld
*@Author 白芨
*/

2、标识符

Java所有组成部分都需要名字，类名、变量名以及方法名都称为标识符。

  • 所有标识符应该以字母（ A-Z 或者 a-z ）美元符（ $ ）下划线( _ ) 开始
  • 不能使用关键字作为变量名、方法名
  • 标识符大小写敏感

3、数据类型

强类型语言

  • 要求变量使用严格符合规定，所有变量要求先定义才能使用

基本类型（primitive type）

八大基本数据类型

数值类型
整数类型 byte占1个字节； short占2个字节； int占4个字节； long占8个字节,long类型要在数字后加L;
浮点类型 double占8个字节 float占4个字节,float类型要在数字后加F；
字符类型 char占2个字节
Boolean
占1位，其值只有true和false两个。

补充：

位（bit）：计算机内部数据存储最小单位

字节（byte）：计算机数据处理基本单位，习惯用大写B表示。

1B（byte，字节）=8bit（位）

引用类型（reference type）

  • 类
  • 接口
  • 数组
//整数扩展：进制表示： 二进制0b		八进制0		十进制		十六进制0x
//浮点数扩展：有限 离散 舍入误差 接近但不等于；最好完全避免使用浮点数进行比较
//字符扩展：所有字符本质还是数字；Unicode编码表：两个字节 0-65536；表示：\u0061 = a

类型转换

Java是强类型语言，运算时需将不同类型数据转换为同一类型然后进行运算。

优先级低 -> 高：byte、short、char、-> int -> long -> float -> double

  • 强制类型转换：(类型)变量名 高->低
  • 自动类型转换

​ 1、不能对布尔值转换

​ 2、不能把对象类型转换为不相干类型

​ 3、转换时可能存在内存溢出，或精度问题

4、变量、常量

  • Java每个变量必须声明其类型
  • Java变量是程序中最基本的存储单元，其类型包括变量名、变量类型、作用域
  • 每个变量都有类型，可以是基本类型、也可以是引用类型
  • 变量名必须是合法标识符
  • 变量声明是一条完整语句，以分号;结束。

变量：

//局部变量,写方法中；只在对应方法中生效。
		//main方法
    public static void main(String[] args) {
        //局部变量，必须声明和初始化值
        int i = 10;
        System.out.println(i);
    }
//---------------------------------------
//实例变量：写在类之内，方法之外。从属于对象；
/*如果不自行初始化，输出对应类型的默认值；
	数值：0 0.0 
	布尔：false
	除基本类型其余默认值：null
*/
//----------------------------------------
//类变量：static
//从属于类

命名规范：

  • 所有变量、方法名、类名：见名知意
  • 类成员变量、类名、方法名：首字母小写驼峰原则monthSalary
  • 常量：大写字母和下划线MAX_VALUE

常量：

//常量 初始化后不能再改变，一般使用大些字符
//final 修饰符，不存在先后顺序
final 常量名 = 值；
final double PI = 3.14；

5、运算符、字符串连接符

运算符

  • 算数运算符
/* -------------------------------------
	加、减、乘、除、取余（模运算）、自增、自减；
	‘+’ 、 ‘-’ 、 ‘*’ 、’/‘（注意类型）、‘%’ 、‘++’、‘--’；
	--------------------------------------
*/
//例：’a++‘(先执行后自增)、‘++a’(先自增后执行)
//幂运算(java 中无 '^'; 需借助工具类如下2^3表示：)
    double pow = Math.pow(2,3);
    System.out.println(pow);

  • 赋值运算符 =

  • 关系运算符 '>'、'<'、'>='、'<='、‘==’、'!='

/*-----------------------------------
大于、小于、大于等于、小于等于、等于、不等于
'>'、'<'、'>='、'<='、‘==’、'!='
 ------------------------------------
*/
  • 逻辑运算符’&&(与)‘、 ’||（或）‘、 ‘!（非）’
  • 位运算符
/*----------------------------------------------------------
与、或、异或（相同为0，不同为1）、取反、左移(2<<3 == 2*2*2*2)、右移
'&'、'|'、 '^'、 '~'、'<<'、‘>>’
 -----------------------------------------------------------
*/
/*--------------
例：
A = 0011 1100
B = 0000 1101
  
A&B = 0000 1100
A|B = 0011 1101
A^B = 0011 0001
~B = 1111 0010
---------------
*/
  • 条件运算符 
//例：
//x ? y : z;
//x 若为true则结果为y；false则结果为z；
  • 扩展赋值运算符（a+=b）等于 a=a+b

字符串连接符

a = 10
b = 8
System.out.println(""+a+b); //108
System.out.println(a+b+""); //18

6、包机制

  • Java包机制，用于区别于类名的命名空间。
  • 包语句语法格式：
package pk1[.pk2[.pk3]]				//定义包
  • 一般利用公司域名倒置作为包名：com.baidu.www
  • 使用import语句导入包，可使用该包的成员
import pk1[.pk2[.pk3]]				//导入包

7、JavaDoc生成文档

  • javadoc命令用来生成自己的API文档
  • 参数信息
    • @author 作者名
    • @version 版本号
    • @since 指明需要最早使用的jdk版本
    • @param 参数名
    • @return 返回值情况
    • @throws 异常抛出情况
